# Backend

## Статус

Папка `backend` сохранена только как исторический референс ранней версии проекта на `NestJS`.

## Важно

- текущая целевая архитектура проекта: `Next.js only`;
- отдельный backend больше не используется для деплоя;
- весь актуальный API-слой переносится и поддерживается в `frontend/src/app/api/v1`;
- серверная логика находится в `frontend/src/server`.

## Для чего папка пока оставлена

- сверять ранние сервисы и DTO при миграции;
- при необходимости перенести отдельные фрагменты логики;
- иметь архив исходного направления разработки.

## Рекомендация

После полного подтверждения миграции и подключения боевых интеграций папку `backend` можно:

- либо удалить из репозитория;
- либо оставить в архивном виде без участия в CI/CD и деплое.
