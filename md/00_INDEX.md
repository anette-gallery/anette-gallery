# Индекс Документации

В этой папке собраны основные markdown-материалы по проекту на текущий момент.

## Состав папки

- `01_IMPLEMENTATION_PLAN.md` - общий план проекта, архитектура и этапы работ.
- `02_API_CONTRACTS.md` - стартовые API-контракты для `Maxma`, `Tilda` и `1С`.
- `03_REQUIRED_ACCESS_AND_SECRETS.md` - список доступов, секретов и данных, которые нужны для подключения интеграций.
- `04_ROOT_README.md` - общее описание монорепозитория.
- `05_BACKEND_README.md` - краткое описание backend-части.
- `06_FRONTEND_README.md` - краткое описание frontend-части.
- `07_DATABASE_README.md` - краткое описание database-части.
- `08_CURRENT_PROGRESS.md` - сводка того, что уже реализовано в проекте.
- `09_OPEN_QUESTIONS_FOR_CLIENT.md` - открытые вопросы клиенту и зафиксированные ответы.
- `10_DEPLOY_GITHUB_VERCEL.md` - краткая памятка по публикации в `GitHub` и деплою `frontend` в `Vercel`.

## Как использовать

Если позже нужно будет быстро вернуться к проекту, начинать удобнее в таком порядке:

1. `08_CURRENT_PROGRESS.md`
2. `01_IMPLEMENTATION_PLAN.md`
3. `02_API_CONTRACTS.md`
4. `03_REQUIRED_ACCESS_AND_SECRETS.md`
