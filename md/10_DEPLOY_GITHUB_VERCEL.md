# Деплой через GitHub и Vercel

См. основной документ:

- `docs/DEPLOY_GITHUB_VERCEL.md`

Кратко:

1. Создать пустой репозиторий на `GitHub`
2. Выполнить первый `push`
3. Импортировать репозиторий в `Vercel`
4. Выбрать `Root Directory = frontend`
5. Добавить `NEXT_PUBLIC_API_BASE_URL`
6. Развернуть `backend` отдельно
7. После этого подключать `Tilda`, `Maxma` и `1С`
